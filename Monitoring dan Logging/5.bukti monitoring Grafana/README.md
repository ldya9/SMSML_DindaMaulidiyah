# Bukti Monitoring Grafana

Folder ini berisi screenshot dashboard Grafana dengan monitoring metrics.

## Screenshot yang Diperlukan:

1. **1.monitoring_http_requests_total.png** - Panel untuk `http_requests_total`
2. **2.monitoring_system_cpu_usage.png** - Panel untuk `system_cpu_usage`
3. **3.monitoring_system_ram_usage.png** - Panel untuk `system_ram_usage`
4. **4.monitoring_api_latency_seconds.png** - Panel untuk `api_latency_seconds` (untuk Skilled)
5. **5.monitoring_throughput_per_minute.png** - Panel untuk `throughput_per_minute` (untuk Skilled)

## PENTING:

- **Nama dashboard harus berisi username Dicoding Anda**
  - Contoh: `dashboard-dindamaulidiyah` atau `dashboard-ldya9`
- Pastikan nama dashboard terlihat jelas di screenshot
- Screenshot harus menampilkan seluruh dashboard dengan semua panel

## Cara Mengambil Screenshot:

1. Buka Grafana di http://127.0.0.1:3000
2. Login (admin/admin)
3. Buka dashboard yang sudah dibuat (dengan nama username Dicoding)
4. Pastikan semua panel metrics terlihat
5. Screenshot seluruh dashboard
6. Simpan dengan nama sesuai di atas
