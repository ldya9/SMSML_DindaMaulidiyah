Nama: Dinda Maulidiyah
Proyek: Eksperimen SML - Preprocessing Otomatis

Deskripsi Workflow:
Workflow ini dirancang untuk melakukan proses otomatisasi (Continuous Integration)
pada tahap preprocessing dataset menggunakan GitHub Actions.

Trigger:
- Setiap kali ada push atau pull request ke branch "main"

Langkah-langkah yang dilakukan:
1. Setup environment Python 3.12
2. Install dependencies dari requirements.txt
3. Menjalankan script automate_DindaMaulidiyah.py untuk preprocessing dataset
4. Menyimpan output dataset baru (hasil preprocessing)

Tujuan:
Agar setiap perubahan kode preprocessing dapat dijalankan otomatis tanpa harus
dijalankan manual di Google Colab.
